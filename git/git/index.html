把你的git仓库跑起来

发表于 2016-12-20   |  更新于 2016-12-20   |   分类于 git   |  

在git上传了网站。怎么把它跑起来呢？现在已经不需要找个远程服务器了，gitpages帮我们解决了这个问题。

实现方法

将项目push后，运行
git subtree push –prefix=dist origin gh-pages
dist 是要发布的目录
gh-pages 默认网站分支。可以在setting 改为master

就可以把 dist目录下的代码跑起来。

展示

在浏览器输入
你的git账号+github.io，例如 https://daydayupsnail.github.io

加上你的仓库地址 例如react-practice-gallery

即下面的完整地址，就可以访问了react-practice-gallery仓库下dist目录的代码了
https://daydayupsnail.github.io/react-practice-gallery
